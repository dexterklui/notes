## Input elements

### Input attributes

| Attribute     | Description                         |
| ------------- | ----------------------------------- |
| `type`        | Like `text`                         |
| `name`        | Name to identify this input element |
| `placeholder` | Background text as guidelines       |
| `value`       | Value                               |
| `required`    | Required for submission             |
